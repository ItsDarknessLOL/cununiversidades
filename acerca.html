Acerca de este sitio

Este proyecto fue desarrollado con el objetivo de facilitar el acceso a información educativa para estudiantes, padres de familia y cualquier persona interesada en conocer las universidades disponibles en Cancún.

El sitio utiliza tecnologías web modernas como HTML, CSS, JavaScript y Leaflet para ofrecer una experiencia visual clara, rápida y adaptable a dispositivos móviles.

🎓 Enfoque educativo

Información clara y accesible sobre instituciones educativas.

📱 Diseño adaptable

Optimizado para computadoras, tablets y teléfonos móviles.

🌐 Datos abiertos

Basado en información pública y organizada para su consulta.
